Welcome to Indicium!

Indicium offers interactive querying on Data-Platform for Snapdeal transactional and front-end events.
Users can create and share documents (called as 'notebooks') that contain live code, visualizations and explanatory-text.
The notebooks created corresponds to previous 90 days of data for any event by default
